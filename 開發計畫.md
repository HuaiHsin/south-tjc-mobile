| 決策 | 結論 | 影響 |

## 4. `links.json` 資料結構（草案）

```json
{
  "meta": {
    "site": "south.tjc.org.tw",
    "updated": "2026-06-12",
    "categories": [
      { "id": "sabbath",   "name": "安息日與聚會" },
      { "id": "official",  "name": "公文與公告" },
      { "id": "resource",  "name": "教材與資源" },
      { "id": "pastoral",  "name": "教牧與行政" },
      { "id": "media",     "name": "影音與靈糧" },
      { "id": "service",   "name": "服務與表單" }
    ]
  },
  "announcements": [
    { "title": "最新公告標題", "url": "https://south.tjc.org.tw/...", "date": "2026-06-10" }
  ],
  "quickAccess": [
    { "title": "安息日表", "url": "https://south.tjc.org.tw/...", "icon": "calendar" }
  ],
  "links": [
    {
      "id": "uniq-001",
      "category": "sabbath",
      "title": "六月份安息日領會表",
      "url": "https://south.tjc.org.tw/...",
      "type": "doc",          // doc | pdf | excel | page | youtube | external
      "updated": "2026-06-01"
    }
  ]
}
```

> 設計重點：分類用 `id` 引用避免改名連動；`type` 供前端顯示對應圖示；日後更新只改此檔。

## 5. 風險與緩解

| # | 風險 | 衝擊 | 緩解 |
|---|------|------|------|
| R1 | 爬取漏抓 / 動態載入連結 / 需登入區塊 | 連結遺失（違反「功能零遺失」驗收） | A2 出盤點報告交叉比對；E4 做連結回歸驗證；保留人工補登欄位 |

| R3 | LOGO 無法乾淨向量化 | 視覺品質 | 退用高解析去背 PNG；favicon 仍可產出 |

| R5 | 繁中字體在 Android 顯示差異 | 易讀性 | C1 階段多機型驗證系統字 fallback |
