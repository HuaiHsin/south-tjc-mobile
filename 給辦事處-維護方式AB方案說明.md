# 網站維護方式說明（A、B 兩案，請辦事處選擇）

> 對象：辦事處承辦同工。
> 背景：新版手機網站已上線一段時間。目前的做法是「雙軌」——您照舊維護舊網站，
> 志工再用同步工具把內容帶到新版。這份說明提出兩個未來方向，請您評估哪一種比較合適。
> 日期：2026-07-02

---

## 方案 A：維持現況（照舊改網頁＋同步工具）

**您的日常工作，跟現在完全一樣：**

1. 用 Word 修改舊網站的網頁（加連結、排版面）
2. 把 PDF、Word 檔上傳到主機的資料夾（01、02……）
3. 改完後，雙擊桌面的「**同步工具**」，新版網站就會自動跟上

**優點**
- 不用改變任何習慣，現在會做的照做就好

**缺點**
- 同一件事其實做了兩次：既要上傳檔案，又要改網頁加連結
- Word 排版容易跑掉，網頁改壞了要重弄
- 舊網站的系統越來越舊，總有一天會故障，到時仍得換做法
- 網址會顯示「不安全」提醒（瀏覽器的鎖頭是打開的）

---

## 方案 B：新做法（只放檔案，不再改網頁）⭐ 建議

**您的日常工作變成只有兩步：**

1. 把檔案放進對應的資料夾——資料夾名稱就是您熟悉的那些：
   `01-安息日領會表`、`02-南區公文`、`20-高屏小區公文`……
   檔名也照現在的習慣取，例如 `115年8月.pdf`、`兒童音樂營(高雄)--115.7.12.pdf`
2. 雙擊桌面的「**更新網站**」，看到綠色的「✓ 已更新」就完成了
   （約 1～2 分鐘後，網站上就看得到）

**「本月公告」的文字**：用記事本打開 `本月公告.txt`，直接改字、存檔，
一樣按「更新網站」即可。**不用再排 Word 版面。**

**從此不再需要：**
- 改 Word 網頁、貼連結
- 擔心版面跑掉、網頁改壞

**優點**
- 步驟減半：放檔案 → 按一下，就上線
- 網站全面加密（瀏覽器顯示安全鎖頭 🔒，不再有「不安全」警告）
- 每一次更新都有紀錄：放錯檔案、不小心刪掉，都可以完整救回來
- 每次更新後，志工的手機會自動收到通知，有問題馬上有人看到
- 不再依賴老舊的主機系統

**需要配合的事**
- 檔名照慣例命名（就是您現在的命名方式；取錯也不會不見，只是標題顯示得比較陽春）
- 第一次需要志工到場（或遠端）幫電腦做一次性設定，並陪您操作一、兩次
- 剛開始一個月是「試營運」：舊網站先不關，兩邊並行，確認新做法順了才收

## 一頁比較

| | 方案 A（照舊） | 方案 B（新做法）⭐ |
|---|---|---|
| 每次更新的步驟 | 上傳檔案＋改 Word 網頁＋跑同步 | **放檔案＋按一下** |
| 要學新東西嗎 | 不用 | 一次教學（約半天）＋練習一兩次 |
| 排版會跑掉嗎 | 會（Word 網頁的老問題） | 不會（版面是固定的，只認檔案） |
| 放錯／誤刪救得回嗎 | 難 | **可以，每次更新都有紀錄** |
| 網址安全鎖頭 🔒 | 沒有（顯示「不安全」） | **有** |
| 舊主機壞掉的風險 | 一直存在 | 不受影響 |
| 費用 | 現有主機費用照付 | **免費**（用 GitHub 免費服務） |

## 常見問題

**Q：舊網站會馬上關掉嗎？**
不會。選了 B 也會先並行至少一個月，一切順了才逐步收掉；要關之前一定會先跟您確認。

**Q：我放錯資料夾或刪錯檔案怎麼辦？**
再按一次「更新網站」放回正確位置就好；真的找不回來，志工可以從紀錄裡把任何時間點的檔案救回來。

**Q：檔案很大（例如錄音檔）放不上去怎麼辦？**
「更新網站」會直接提醒您這個檔太大，先跳過它、其他照常更新，大檔交給志工另外處理。

**Q：按「更新網站」時斷網或當掉？**
不會弄壞任何東西。等網路恢復後再按一次就好。

**Q：以後想新增一個新的分類（新資料夾）？**
可以，跟志工說一聲，加好對照後您就能照常放檔案。

這次盤點時發現，網站上有 **127 個連結**點下去是「找不到檔案」——因為舊主機上的檔案早已被搬走。
無論選 A 或 B，志工都會把這批失效連結清理掉，讓網站上點得到的都是有效的。
