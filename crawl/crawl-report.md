## 2. 關鍵技術發現（影響開發決策）

| # | 發現 | 影響 / 對策 |
|---|------|------------|
| **F1** | **HTTPS 失效**：`https://south.tjc.org.tw` SSL 握手被重置（ERR_CONNECTION_RESET）；只有 **HTTP** 正常回 200 | 規格寫的是 https，但實際只有 http 可用。新站連結需指向 `http://`，或先確認辦事處是否會修好憑證。**建議向業主回報。** |
| **F2** | **整站是 Microsoft Word 匯出的 HTML**，編碼 **big5**，Server 為 IIS 10 / ASP.NET | 無響應式、版面是 Word 表格排版（規格問題的根因）。爬取需 big5 解碼。新站全部改 UTF-8。 |
| **F3** | 子頁採編號目錄結構：`01/` ～ `20/`，多數為 `index.htm`，少數用中文檔名（`06/讀經教材.htm`、`09/人力資源.htm`）或多頁（`03/index1`、`18/index1~4`）| 連結指向原 URL 即可，路徑不變（符合規格「下載檔位置不變」）。 |
| **F4** | 伺服器同時接受 **UTF-8 percent-encoded** 的中文 URL（非只 big5） | 新站連結用標準 UTF-8 編碼即可正常開啟。 |
| **F5** | 首頁有 3 個副本：`/`、`/index.htm`、`/index2.htm` 內容幾乎相同 | 取 `/` 為準即可。 |

## 4. 需業主確認的疑義（A3/A4 用）

1. **HTTPS 是否修復**（F1）— 決定新站連結用 http 或 https。
2. **「18 南區AI指引」歸類** — 規格六大分類無對應；歸入「教材與資源」或新增分類？
3. **「19 真光園區」性質** — 是場地/營會/園區介紹？影響歸類。
4. **「03 教會連結」** — 是各教會聚會時間表，還是純地址清單？影響歸「安息日與聚會」或「教牧」。
