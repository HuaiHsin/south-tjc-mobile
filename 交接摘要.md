# 交接摘要 — 真耶穌教會南區辦事處官網（手機/桌機優化版）

## 一句話
把 `http://south.tjc.org.tw`（Word 匯出、big5、無響應式的舊官網）重製成手機/桌機優化的靜態站，連結全部指回原站檔案，**已上線 GitHub Pages**。

## 線上位置
- **DEMO（給辦事處看）**：https://huaihsin.github.io/south-tjc-mobile/
- **Repo**：https://github.com/HuaiHsin/south-tjc-mobile （public，owner `HuaiHsin`，`main` 分支，Pages = main/root）
- 本地預覽：`python -m http.server 8900` → http://127.0.0.1:8900/index.html
  - ⚠️ 瀏覽器會快取 css/js，看新版要 **Ctrl+F5**，或開 `?v=時間戳` 繞過。

## 技術
純 HTML + CSS + 原生 JS，零依賴、零建置。`fetch` 讀 JSON 渲染。

## 檔案地圖
```
index.html                  單頁首頁
css/styles.css              設計 token + 全部樣式（品牌色：寶藍#0041C4 / 青綠#00B2A2 / 白 / #333）
js/app.js                   載入資料、渲染、互動（下拉選單、詳情overlay、本月公告…）
js/search.js                站內搜尋過濾
data/links.json             四大菜單 + 群組 + quickAccess（單一資料源）
data/announcements.json     本月公告（首頁公告區塊，含純文字）
data/churches.json          各教會通訊錄（名稱+相片/簡介/FB+地址電話）
assets/                     飛鳥LOGO PNG + 多尺寸 favicon
crawl/crawl.py              爬全站連結 + 各頁標題
crawl/churches.py           解析各教會通訊錄
crawl/categorize.py         歸入四大菜單（含排序、schedule、安全網）★核心對照表在這

sync.py / sync.bat          一鍵同步（志工跑這個 / 雙擊 bat）
README.md                   使用與部署說明
```

- **首頁順序**：①AI 靈修智能體（置頂）→ ②Hero（特別公告＋尋人啟示）→ ③教區活動橫條 → ④本月公告（最新公告/宣導/備查 + 任何新區塊）→ ⑤資訊分類（4 張卡）→ ⑥頁尾。
- **分類詳情**（點菜單/卡片開全屏 overlay，有返回堆疊）：群組清單；安息日領會表=12月曆格（W/P並排、空月留白）；教會地址=高屏/台南兩按鈕→教會卡；大群組「查看全部」分頁。
- **頁尾**：4 欄（站名 / 網站地圖 / 聯絡奉獻含複製鈕 / 實用連結）+ 版權。

## 半自動同步（維護模式 C：先 wrapper，日後再做編輯器）
辦事員照舊改 Word 舊站 → 志工跑 `python sync.py`（crawl→churches→categorize→home_bulletin，含備份/失敗還原/門檻檢查）→ `git push` → Pages 自動更新。
- ✅ 自動：既有結構內的內容增刪改。
- ✅ 自動：**首頁新增公告區塊**（任何 `XXX：` 大標）→ home_bulletin 通用偵測 + app.js 自動顯示。
- ⚠️ 需補一行：**新增編號子頁/新菜單** → 安全網用頁標題建群組、暫放「其他資源」最底、sync 印提醒；要正確歸類在 `categorize.py` 的 `SOURCE_MAP` 加一行。
