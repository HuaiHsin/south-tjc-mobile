## 一句話
把 `http://south.tjc.org.tw`（Word 匯出、big5、無響應式的舊官網）重製成手機/桌機優化的靜態站，連結全部指回原站檔案，**已上線 GitHub Pages**。

## 線上位置
- **DEMO（給辦事處看）**：https://huaihsin.github.io/south-tjc-mobile/
- **Repo**：https://github.com/HuaiHsin/south-tjc-mobile （public，owner `HuaiHsin`，`main` 分支，Pages = main/root）
- 本地預覽：`python -m http.server 8900` → http://127.0.0.1:8900/index.html
  - ⚠️ 瀏覽器會快取 css/js，看新版要 **Ctrl+F5**，或開 `?v=時間戳` 繞過。

## 技術
純 HTML + CSS + 原生 JS，零依賴、零建置。`fetch` 讀 JSON 渲染。

## 半自動同步（維護模式 C：先 wrapper，日後再做編輯器）
辦事員照舊改 Word 舊站 → 志工跑 `python sync.py`（crawl→churches→categorize→home_bulletin，含備份/失敗還原/門檻檢查）→ `git push` → Pages 自動更新。
- ✅ 自動：既有結構內的內容增刪改。
- ✅ 自動：**首頁新增公告區塊**（任何 `XXX：` 大標）→ home_bulletin 通用偵測 + app.js 自動顯示。
- ⚠️ 需補一行：**新增編號子頁/新菜單** → 安全網用頁標題建群組、暫放「其他資源」最底、sync 印提醒；要正確歸類在 `categorize.py` 的 `SOURCE_MAP` 加一行。

## ★重製內頁（美編整頁，2026-06-18~19 大批補做）
**背景**：最初爬蟲只抓 `<a>` 連結 + 歸菜單，把原站表格欄位與純文字說明全丟了。後續逐頁重抓、重新美編成「整頁」，點菜單/分類置頂卡開全屏 overlay（沿用返回堆疊）。

**兩種實作路線**
1. **客製整頁**（各有專屬 render + CSS）：南區AI指引、傳道影音專區、校園團契、信徒服務。資料在 `data/*.json`，render 函式在 `js/app.js`。
2. **通用表格頁 `renderDocTable`**（一個元件吃所有表格頁）：桌機條紋表格／手機每列轉卡片（`thead` 隱藏，`td` 用 `data-label` 顯欄名）。儲存格支援單連結 `{t,u,k}`、多連結 `{t,parts:[…]}`→chips、純文字。

**新增表格頁只要兩步**：
- 寫 `crawl/doc_xxx.py` 解析原頁 → 產生 `data/doc-xxx.json`（schema：`{title,subtitle,kicker,channel?,note?,sections:[{name,columns,primary,rows}]}`）。
- 在 `js/app.js` 最上方 `DOC_DEFS` 陣列加一行 `{gid, file, icon, sub}`（`gid` = `links.json` 群組 id，如 `official|南區公文`）。
- 分類置頂卡入口 + 路由（`openGroup`→`DOCS[gid]`）會自動接上。

**已重製/補回的頁面（15 個）**
- 客製：南區AI指引(18)、傳道影音專區(13)、校園團契(04)、信徒服務(10，40 家商家)
- 表格頁：南區公文(02)、高屏小區公文(20)、講義彙編(08，186 則四分類)、教牧資源分享(12，依股別)、健康醫訊(16)、真光園區(19)、生活資訊(14)、聖樂活動影音(05)、宗教教育週海報(09)、線上靈糧(15)、好站連結(11)、讀經教材(06)
- 群組級補字：安息日領會表加經文引言、家護園地加致謝（`links.json` 群組可加 `verse`/`note` 欄位，render 自動顯示）

**注意**
- 這些 `data/doc-*.json` 是**手動跑 `crawl/doc_*.py` 更新**，`sync.py` 目前**不含**它們。若原站對應頁改了，要手動重跑該 `doc_*.py`（之後可考慮接進 sync）。
- 原 `links.json` 群組保留不動 → 站內搜尋仍能搜到這些連結。
- 抓原站務必用 **urllib（Mozilla UA）**，`curl` 對部分頁（如 02）會拿到截斷版。`06` 正確檔名是 `讀經教材.htm` 不是 `index.htm`。
