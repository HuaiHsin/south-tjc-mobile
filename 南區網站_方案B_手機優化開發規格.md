# 真耶穌教會南區辦事處官網 — 手機優化開發規格（方案B：分類重構）

> 交付對象：Claude Code
> 目標網站：https://south.tjc.org.tw
> 原則：**所有現有功能與連結 100% 保留**，僅重構前端架構與資訊分類，不動既有內容資料與後台。

---

## 1. 專案目標

| 項目 | 現況 | 目標 |
|------|------|------|
| 版面 | 純文字連結堆疊、無分區 | 分類卡片化、層次清楚 |
| 行動裝置 | 無響應式、字小難點 | 手機優先（mobile-first） |
| 找資訊 | 需肉眼掃整頁 | 分類 + 站內搜尋 + 常用置頂 |
| 導覽 | 無導覽列 | 固定頂部列 + 漢堡選單 |

**範圍限制**：純前端重構。後台 / CMS / 檔案儲存路徑維持不變，所有連結指向原 URL。

---

## 2. 資訊架構（重點工作）

把首頁所有零散連結重新歸納為 **6 大分類**。開發前需先完整盤點現有頁面所有連結，逐一歸位（見 §8 待確認）。

### 分類定義

1. **安息日與聚會**
   - 安息日領會表（各月 W檔 / P檔）
   - 南區各教會聚會時間表、早禱會時間表
   - 靈恩會安排表

2. **公文與公告**
   - 南區公文、高屏小區公文
   - 最新公告、教區活動預告
   - 總會宣導事項、備查公告、特別公告

3. **教材與資源**
   - 讀經教材、講義彙編
   - 宗教教育海報、宗教教育資源
   - 讚美詩歌詞投影系統（1-150 / 151-310 / 311-469）
   - 聖經章節投放軟體下載

4. **教牧與行政**
   - 駐牧傳道安排表、教會負責人名單
   - 人力資源、教牧資源分享
   - 總會電話分機表

5. **影音與靈糧**
   - 聖樂活動影音 / YouTube 入口
   - 傳道靈糧分享、線上靈修
   - 健康、家護園地、校園團契

6. **服務與表單**
   - 受洗 / 除偶像等申請表單
   - 志工服務證明、學生住宿資訊
   - 信徒服務網頁、好站連結、郵政劃撥、捷運路線圖

> ⚠️ 上述歸類為建議草案，開發時須與業主（南區辦事處）確認後定案。

## 3. 頁面結構

### 首頁（單頁式）
```
[固定頂部列] LOGO ＋ 站名 ＋ 漢堡選單 ＋ 搜尋圖示
[置頂橫幅]   最新公告（1～3則，可橫向滑動）
[常用快捷]   4～6 個大按鈕：安息日表｜聚會時間｜最新公文｜讚美詩投影
[分類區塊]   6 大分類卡片，點擊展開該類連結清單（折疊式 accordion）
[頁尾]       聯絡信箱、劃撥帳號、捷運圖、版權
```

### 搜尋
- 站內關鍵字即時過濾（前端 JS，比對連結標題與分類）
- 不需後端搜尋引擎，資料量小用 client-side filter 即可

## 5. 設計規範（手機優先）

```
斷點：  手機 <768px（主要）｜平板 768-1024｜桌機 >1024
字級：  內文 16px、標題 20-24px、最小 14px（禁用 12px 以下）
觸控：  可點區域最小 44×44px，連結間距 ≥8px
字體：  系統字（-apple-system, "PingFang TC", "Microsoft JhengHei", sans-serif）
        確保繁體中文清晰
間距：  區塊間 24px、卡片內 16px、留白充足
```

### 品牌色票（取自官方 LOGO，精確值）

| 用途 | 色名 | HEX |
|------|------|-----|
| 主色 | 寶藍（飛鳥主體） | `#0041C4` |
| 輔色 | 青綠（飛鳥尾翼／標準字） | `#00B2A2` |
| 主背景 | 純白 | `#FFFFFF` |
| 內文 | 深灰 | `#333333` |
| 次要文字／邊框 | 淺灰 | `#E5E5E5` |

```css
:root{
  --color-primary:#0041C4;   /* 寶藍 */
  --color-accent:#00B2A2;    /* 青綠 */
  --color-bg:#FFFFFF;
  --color-text:#333333;
  --color-border:#E5E5E5;
}
```

> 配色原則：寶藍為主導（頂部列、主按鈕、標題），青綠為點綴（badge、強調、hover），大面積留白。莊重清楚，符合教會性質。

### LOGO 資源
- `真耶穌教會全球通用LOGO`：飛鳥圖標，用於頂部列、favicon、載入畫面。
- `真耶穌教會LOGO標準字`：橫式「飛鳥＋真耶穌教會」字標，用於頂部列站名、頁尾。
- 原檔為 CMYK `.tif`，**開發前須轉為網頁用 RGB 格式**：圖標另存 `.svg`（向量優先）或去背 `.png`，並產出多尺寸 favicon。

設計調性：**莊重、清楚、易讀**，避免花俏動畫。

## 6. 技術建議

- **框架**：靜態網站。建議純 HTML + CSS + 原生 JS，或輕量框架（Astro / Vite）。資料量小，避免過度工程。
- **資料來源**：將所有連結整理成單一 `links.json`（分類、標題、URL、更新日期），前端讀取渲染。日後更新只改 JSON。
- **連結策略**：所有連結指向原 `south.tjc.org.tw` 既有 URL，下載檔（W檔 / P檔 / Excel 軟體）儲存位置**不變**，僅前台重新呈現。
- **部署**：**先部署至 GitHub Pages 作為 DEMO**，供辦事處審閱。Repo 建議 `south-tjc-mobile`，啟用 Pages（main 分支 / root 或 `/docs`）。確認後再決定是否切換正式網域。
- **相容性**：支援 iOS Safari / Android Chrome，向下相容至近 3 年版本。
- **效能**：首頁載入 <2 秒，圖片延遲載入。
